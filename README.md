# JavaScript CLI Tool

A fancy JavaScript CLI tool for millionaire game.

## Installation

```bash
git clone git@github.com:HUIXIN-TW/node_cli.git
```

## How to run

```bash
node index.js
# or use #!/usr/bin/env node
node .
# or use #!/usr/bin/env node
./index.js
```

Remember to give the file permission to execute if using `./index.js`.

```bash
chmod +x index.js
```
